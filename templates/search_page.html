Using ML for Cardiovascular Diseases

This endpoint contains 4 trained models covering 3 different tasks: catheterization prediction, EF prediction, and followup EF prediction. The tabs above take you to the models: in each tab, there is a form which asks for entries (the data input), and upon submitting, you will get the prediction from the corresponding model.
